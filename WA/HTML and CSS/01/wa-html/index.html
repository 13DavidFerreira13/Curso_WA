Estrutura do HTML

Estrutura do HTML
  • Hyper Text Markup Language é uma linguagem de marcação de elementos e conteúdo e serve para estruturar páginas web.
  • HTML é um formato de texto simples
  • Elementos são constituídos por tags (etiquetas)
  • Existem tags para texto, imagem, vídeo, entre outras
  • Browsers apenas mostram o conteúdo contido nas tags

<meta charset="utf-8">

  • Contém o encoding de caracteres da página
  • Unicode Transformation Format (UTF)
  • Desenvolvido pelo Unicode Consortium
  • É um standard nos formatos HTML, XML, Java, JavaScript, E-mail, ASP, PHP e SOs.

Anatomia de uma tag

  • Elementos são normalmente definidos por duas tags (abertura e encerramento), conteúdo e propriedades
  • O nome do elemento escreve-se entre < e >
  • Tag de abertura e encerramento tem o mesmo nome
  • Tag de encerramento destaca-se pela precedência de uma barra inclinada ao nome do elemento

Hiperligações

Hiperligações

<a>

  • Tag <a> define uma hiperligação (anchor)
  • Links internos ou externos
  • Caminho (path) relativo ou absoluto
  • Azul, indica links não visitados
  • Roxo e sublinhado indica links visitados
  • Vermelho e sublinhado indica link activo
  • Atributo href define url de destino
  • Atributo target define janela de destino
